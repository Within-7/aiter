# Changelog

All notable changes to AiTer will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-12-03

### Added
- 自动检查更新功能
- 版本发布页面生成
- 应用内更新通知
- 多项目管理功能
- 多终端标签支持
- HTML 文件实时预览
- Monaco 代码编辑器集成
- Markdown 预览功能
- 插件系统支持
- 项目文件浏览器

### Features
- 优化 AI CLI 工具协作体验
- 支持 Minto CLI、Claude Code CLI、Gemini CLI
- 跨平台支持 (macOS & Windows)
- 内置文件服务器
- 自定义终端 Shell 配置

### Infrastructure
- Electron 28 架构
- React 18 + TypeScript
- xterm.js 终端模拟
- Monaco Editor 代码编辑
- 自动更新系统

# AiTer 团队下载使用指南

欢迎使用 AiTer！本指南将帮助团队成员快速下载和安装 AiTer 应用程序。

## 📥 快速下载

### 第一步：访问 Releases 页面

点击以下链接访问最新版本下载页面：

```
https://github.com/Within-7/aiter/releases/latest
```

或者：

1. 访问 GitHub 仓库：`https://github.com/Within-7/aiter`
2. 点击右侧的 **Releases** 标签
3. 选择最新版本（通常在最顶部）

### 第二步：选择对应平台的安装包

根据你的操作系统选择下载文件：

#### 🍎 macOS 用户

**如何确认你的 Mac 芯片类型：**
1. 点击屏幕左上角的  菜单
2. 选择"关于本机"
3. 查看"芯片"或"处理器"信息

**下载对应文件：**
- **Apple Silicon** (M1/M2/M3/M4 芯片)
  - 文件名：`AiTer-{版本号}-mac-arm64.dmg`
  - 示例：`AiTer-0.1.0-mac-arm64.dmg`

- **Intel 芯片** (i5/i7/i9)
  - 文件名：`AiTer-{版本号}-mac-x64.dmg`
  - 示例：`AiTer-0.1.0-mac-x64.dmg`

#### 🪟 Windows 用户

- 文件名：`AiTer-{版本号}-win-x64.exe`
- 示例：`AiTer-0.1.0-win-x64.exe`

## 💿 安装步骤

### macOS 安装

1. **下载并打开 DMG 文件**
   - 双击下载的 `.dmg` 文件

2. **安装应用**
   - 将 AiTer 图标拖拽到 Applications 文件夹
   - 等待复制完成

3. **首次打开（重要！）**

   由于应用未经过 Apple 公证，首次打开需要特殊步骤：

   **方法一：右键打开（推荐）**
   - 在 Applications 文件夹中找到 AiTer
   - **按住 Control 键点击**（或右键点击）应用图标
   - 选择"打开"
   - 在弹出的对话框中点击"打开"
   - 后续可以正常双击打开

   **方法二：系统设置允许**
   - 尝试双击打开 AiTer
   - 如果看到"无法打开，因为无法验证开发者"提示
   - 打开"系统设置" → "隐私与安全性"
   - 找到"AiTer 已被阻止"提示
   - 点击"仍要打开"
   - 输入管理员密码确认

4. **完成！**
   - 应用现在可以正常使用了

### Windows 安装

1. **运行安装程序**
   - 双击下载的 `.exe` 文件

2. **Windows Defender 提示**

   如果看到"Windows 已保护你的电脑"提示：
   - 点击"更多信息"
   - 点击"仍要运行"

3. **按照安装向导操作**
   - 选择安装位置（默认即可）
   - 点击"安装"
   - 等待安装完成

4. **完成！**
   - 桌面会创建快捷方式
   - 双击打开 AiTer

## 🚀 开始使用

### 首次启动

1. **添加项目**
   - 点击左侧边栏的"+"按钮
   - 选择你的项目文件夹
   - 项目会出现在侧边栏中

2. **打开终端**
   - 点击项目名称展开
   - 点击"New Terminal"按钮
   - 终端会在右侧打开

3. **浏览文件**
   - 点击项目下方的"FILES"
   - 文件树会展开显示项目文件
   - 点击文件可以在编辑器中打开

## ❓ 常见问题

### Q1: 下载需要登录 GitHub 吗？

**是的。** 由于仓库是私有的，你需要：
1. 拥有 GitHub 账号
2. 被授予仓库访问权限

如果无法访问，请联系管理员添加权限。

### Q2: macOS 提示"已损坏"怎么办？

这是因为应用未签名。解决方法：

```bash
# 在终端中运行（替换为实际路径）
sudo xattr -cr /Applications/AiTer.app
```

然后重新打开应用。

### Q3: 如何更新到最新版本？

1. 访问 Releases 页面
2. 下载最新版本
3. 安装新版本（会覆盖旧版本）
4. 你的项目和设置会自动保留

### Q4: 可以安装在多台电脑上吗？

可以。只要你有仓库访问权限，可以在任意台电脑上下载安装。

### Q5: 支持哪些操作系统版本？

- **macOS**: 10.12 (Sierra) 或更高版本
- **Windows**: Windows 10 或更高版本

### Q6: 需要安装 Node.js 吗？

**不需要。** AiTer 是独立应用程序，已包含所有必需组件。

但如果你要使用需要 Node.js 的 AI CLI 工具（如 Minto CLI），则需要单独安装 Node.js。

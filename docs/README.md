## 🚀 快速开始路径

### 新用户入门 (第一天)
```
1. 阅读 [产品网站](index.html) - 5分钟
   了解AiTer是什么,能做什么

2. 阅读 [用户手册 - 快速开始](USER_MANUAL.md#快速开始) - 10分钟
   安装并创建第一个项目

3. 阅读 [用户手册 - 基础操作](USER_MANUAL.md#基础操作) - 15分钟
   学习项目管理和终端操作

4. 实践: 创建测试项目,尝试基本功能 - 30分钟
```

### 咨询团队成员入职 (第一周)
```
Day 1-2: 基础功能掌握
- 完整阅读 [用户手册](USER_MANUAL.md)
- 创建个人练习项目
- 熟悉快捷键

Day 3-4: 工作流学习
- 阅读 [战略咨询工作流指南](CONSULTING_WORKFLOW.md)
- 了解公司标准流程
- 复用模板库

Day 5: 最佳实践
- 阅读 [最佳实践指南](BEST_PRACTICES.md)
- 参加团队知识分享会
- 开始参与真实项目
```

### 高效使用进阶 (持续)
```
1. 定期回顾 [最佳实践](BEST_PRACTICES.md)
   每月一次,发现优化空间

2. 参考 [工作流指南](CONSULTING_WORKFLOW.md)
   项目启动时查阅标准流程

3. 查阅 [用户手册](USER_MANUAL.md)
   遇到问题时查找解决方案

4. 贡献知识
   发现新方法时,更新文档
```
